Select HTML Editor Classes

Flyweight Classes

Character Flyweight

This is a class implementing the flyweight design pattern to store a character value. Each character in a text document is stored as a flyweight object, of which there can be only one per unique character. This technique reduces the amount of space used to store the document.

Flyweight Factory

The flyweight factory is a class which generates flyweight objects. It is fed characters and returns the respective flyweight object or creates a new one if one does not yet exist.

Composite Structure Classes

To keep the text in a file in order a composite tree structure was implemented

Column

The compositorCol class acts as the root of the tree, and imitates the first column of the text document.

Row

The compositorRow class stores a list of leaf objects; one for each character in a row.

Leaf

Leaf objects act as the leafs of the tree and store flyweight characters.

Iterator Classes

Preorder Iterator

This class iterates through the tree in a preorder structure.

List Iterator

This class iterates through the row and column nodes in the composite tree as if they were lists.

Visitor Classes

Text Visitor

This visitor class iterates through the tree using an iterator and ignores any added html tags to a document. It can be used to display the original document without added HTML.

HTML Visitor

The HTML visitor will iterate through the tree using an iterator without ignoring the tags added by the editor. With both of these classes it is ease to switch between viewing the original and edited document.
